# Aktualizacja: dynamiczny szablon projektu i standard plików

Ta paczka dopisuje do projektu sklepu z projektami domów ustalenia dotyczące:

- jednej dynamicznej karty projektu,
- podmieniania danych/opisów/zdjęć/rzutów dla każdego projektu,
- struktury folderów publicznych mediów,
- struktury prywatnych plików po zakupie,
- stałego nazewnictwa plików,
- rozdzielenia danych projektu od obrazów,
- kolejnego etapu wdrożenia.

To jest ważne, żeby sklep nie był zbiorem ręcznie dłubanych stron, tylko systemem opartym o jeden szablon i dane.
