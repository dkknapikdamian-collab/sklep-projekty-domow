# Aktualizacja V8 — admin panel lokalny pod późniejsze Supabase

Ta paczka dodaje produkcyjny szkielet panelu admina bez podłączania jeszcze Supabase.

Cel:
- zobaczyć realny wygląd panelu admina,
- mieć zakładkę dodawania projektu,
- mieć podgląd karty projektu bez publikowania fikcyjnej oferty,
- przygotować UI i strukturę pod późniejsze podpięcie Supabase Auth/Postgres/Storage.

To nie jest fałszywy panel do wyrzucenia. 
To jest produkcyjny UI admina z tymczasowym lokalnym trybem danych.
