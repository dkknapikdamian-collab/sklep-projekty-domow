# Developer handoff

## Co budujemy

Produkcyjny sklep z projektami domów.

## Co jest najważniejsze

1. Zachować wygląd.
2. Nie robić danych na sztywno.
3. Zrobić mocną kartę projektu.
4. Zrobić panel admina.
5. Zapisywać pełne zamówienia.
6. Wysyłać e-maile i logować je.
7. Po płatności wysyłać PDF/link albo oznaczać ręczną obsługę.
8. Wyszukiwarka ma być normalizowana.

## Pułapki

- publiczne media pomylone z prywatnymi PDF,
- wysyłka plików przed płatnością,
- ufanie return URL płatności,
- obowiązkowe konta klienta przed zakupem,
- zbyt dużo filtrów,
- rozjechany styl,
- osobny CSS dla każdego ekranu.
