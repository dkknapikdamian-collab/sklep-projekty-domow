# Checklist odbioru

## Wygląd

- [ ] Styl zgodny z referencją.
- [ ] Ciemny grafit i złoto-beżowe akcenty.
- [ ] Karta projektu ma prawy box zakupowy.
- [ ] Kafle parametrów czytelne.
- [ ] Katalog spójny.
- [ ] Mobile używalny.

## Projekty

- [ ] Projekt ma kod.
- [ ] Projekt ma nazwę.
- [ ] Projekt ma cenę.
- [ ] Projekt ma parametry.
- [ ] Projekt ma pomieszczenia.
- [ ] Projekt ma media.
- [ ] Projekt ma dodatki.
- [ ] Projekt ma pliki prywatne.

## Wyszukiwarka

- [ ] Szuka po kodzie.
- [ ] Szuka po nazwie.
- [ ] Szuka po metrażu.
- [ ] Szuka po działce.
- [ ] Normalizuje polskie znaki.
- [ ] Normalizuje m2/m²/m kw.
- [ ] Rozumie 20x30.
- [ ] Rozumie garaż/garaz.

## Koszyk i checkout

- [ ] Dodaje projekt.
- [ ] Obsługuje wariant.
- [ ] Obsługuje dodatki.
- [ ] PDF_EMAIL_PACKAGE dodaje +250 zł.
- [ ] Suma jest poprawna.
- [ ] Checkout tworzy order i order_items.

## Admin

- [ ] Widzi projekty.
- [ ] Dodaje projekt.
- [ ] Edytuje projekt.
- [ ] Dodaje pomieszczenia.
- [ ] Dodaje media.
- [ ] Dodaje pliki.
- [ ] Widzi zamówienia.
- [ ] Ma filtr opłacone i niewysłane.
- [ ] Ma filtr PDF do wysłania.

## Płatności/e-maile/pliki

- [ ] Webhook paid działa.
- [ ] Failed nie daje dostępu.
- [ ] Potwierdzenia e-mail działają.
- [ ] PDF/link po płatności działa.
- [ ] Email logs działają.
- [ ] Prywatne PDF/ZIP nie są publiczne.
