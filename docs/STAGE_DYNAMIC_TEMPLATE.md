# Etap wdrożony: dynamiczny szablon projektu

## Cel

Przerobić kartę projektu z jednej statycznej strony na dynamiczny szablon obsługujący wiele projektów.

## Zrobione

- `/projekty/[slug]`
- `data/projects.ts`
- minimum 3 projekty demo
- foldery mediów po kodzie projektu
- komponenty:
  - `ProjectDetailPage`
  - `ProjectGallery`
  - `ProjectPurchaseBox`
  - `ProjectStatsStrip`
  - `ProjectTabs`
  - `RelatedProjects`
  - `ProjectCard`
- ścieżki mediów pobierane z `project.media`
- dodatki pobierane z `project.addons`
- PDF_EMAIL_PACKAGE jako dodatek +250 zł
- katalog `/projekty`

## Ważne

To dalej jest frontend demo. Następny etap to dopracowanie katalogu i koszyka/checkoutu na danych dynamicznych, a dopiero potem baza i admin.
