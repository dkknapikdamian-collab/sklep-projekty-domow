# Struktura mediów projektu

## Publiczne media realnego projektu

Każdy prawdziwy projekt ma folder po kodzie:

```txt
public/projects/DP-AUR-014/
```

Zalecane pliki:

```txt
hero.jpg
thumbnail.jpg
gallery-01.jpg
gallery-02.jpg
gallery-03.jpg
gallery-04.jpg
floor-plan-ground.jpg
floor-plan-floor.jpg
floor-plan-roof.jpg
section-aa.jpg
section-bb.jpg
elevation-front.jpg
elevation-garden.jpg
elevation-left.jpg
elevation-right.jpg
```

## Prywatne pliki zakupowe

Nie wkładać do `public`.

Logiczna struktura:

```txt
private/projects/DP-AUR-014/
  documentation-v1.pdf
  full-package-v1.zip
  pdf-email-package-v1.pdf
```

Docelowo to będzie prywatny storage/chmura/serwer z kontrolą dostępu.

## Zasada

Publiczne media pokazują projekt.  
Prywatne pliki są dostępne dopiero po zamówieniu/płatności.
