# Bezpieczeństwo i kwestie formalne

To nie jest porada prawna. Treści regulaminu, polityki prywatności, zgód, reklamacji i odstąpienia powinny być sprawdzone przez prawnika.

## Konieczne technicznie

- regulamin,
- polityka prywatności,
- zgody w checkout,
- zapis wersji zgód przy zamówieniu,
- formularz reklamacji,
- formularz odstąpienia,
- logi e-maili,
- prywatne pliki bez publicznych URL,
- tokeny do pobrań,
- webhook płatności,
- ochrona panelu admina,
- walidacja formularzy,
- rate limit na formularzach,
- backup bazy,
- backup plików,
- brak sekretów w repo.
